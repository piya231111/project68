# StarWorld Project

## ที่มาและเหตุผล
   ในยุคนี้ผู้คนมาจากหลากหลายภูมิหลัง แนวคิด และความสนใจ ต่างเชื่อมโยงด้วยกันผ่านโลกออนไลน์ แต่การสื่อสารบนแพลตฟอร์มทั่วไปยังมีข้อจำกัดที่อาจทำให้บางคนรู้สึกไม่เป็นตัวของตัวเอง หรือไม่กล้าแสดงออกอย่างเต็มที่ อยากจะคุยกลับกลุ่มเพื่อนใหม่ ๆ ที่มีความสนใจคล้ายกัน โครงการนี้พัฒนาขึ้นเพื่อสร้างสรรค์เว็บไซต์คอมมูนิตี้ที่ใช้ตัวละครสมมุติ (Avatar) เป็นสื่อกลางเพื่อสร้างพื้นที่ออนไลน์ที่เปิดกว้างสำหรับทุกคน สามารถแบ่งปัน เชื่อมโยง และแสดงตัวตนในแบบของตัวเองได้อย่างอิสระแลปลอดภัย โดยไม่จำเป็นต้องเปิดเผยตัวตนจริง ส่งเสริมความสัมพันธ์ใหม่ ๆ ในเชิงสร้างสรรค์ และความเพลิดเพลินในการสื่อสาร

## วัตถุประสงค์
 1. เพื่อพัฒนาระบบสนทนาออนไลน์ที่ให้ผู้ใช้สามารถสื่อสารแบบเดี่ยวและแบบกลุ่มได้
 2. เพื่อสร้างระบบจับคู่สนทนาที่ตรงกับความสนใจและประเทศของผู้ใช้
 3. เพื่อพัฒนาระบบโปรไฟล์ที่ปรับแต่งอวตารและข้อมูลส่วนตัวได้
 4. เพื่อจัดทำระบบเพื่อน ดูสถานะออนไลน์–ออฟไลน์ และสื่อสารแบบส่วนตัวได้
 5. เพื่อพัฒนาระบบความปลอดภัย เช่น การตรวจจับคำหยาบ ระบบรายงานผู้ใช้ และการควบคุมโดยผู้ดูแล
 6. เพื่อให้ผู้ใช้สามารถใช้งานระบบได้ง่าย รวดเร็ว และมีประสบการณ์ที่ดี

## ขอบเขตโครงงาน
 1. ผู้ใช้สามารถเลือกตัวละครสำเร็จรูป และเลือกไอเทมพื้นหลังสำหรับตกแต่งตังละครที่มีมาให้ พร้อมเปลี่ยนได้ในภายหลัง
 2. ผู้ใช้สามารถเลือกเข้าห้องแชททั้งแบบแชทกลุ่มและแชทเดี่ยว เพื่อพบเพื่อนใหม่ ๆ ได้
 3. ระบบเพื่อนสำหรับการขอเป็นเพื่อน ฟังก์ชันจัดการเพื่อน เพิ่ม ลบ ปักดาว บล็อค และ แชทกับเพื่อนส่วนตัวได้
 4. ระบบรองรับผู้ใช้หลายคนได้
 5. ผู้ใช้ไม่สามารถปรับสัดส่วนใบหน้าร่างกาย หรือปรับแต่งนอกเหนือจากตัวเลือกสำเร็จรูป และอวาตารจะแสดงผลเป็นภาพนิ่งเท่านั้น
 6. ไม่มีฟังก์ชันการโทรด้วยเสียงหรือวิดีโอ

## Functional Requirements
# ฟังก์ชันหลักของระบบ
1. การจัดการบัญชีผู้ใช้ (User Account Management)
 - เข้าสู่ระบบ และออกจากระบบ
 - ลงทะเบียนผู้ใช้ใหม่
 - เชื่อมต่อบัญชี Google
 - สามารถแก้ไขโปรไฟล์
 - เลือกประเทศ และความสนใจ (จำเป็นต่อระบบจับคู่)
 - เลือกตัวละครสำเร็จรูป และไอเท็มตกแต่ง
2. ห้องแชทแบบเดี่ยว (One-to-One Chat)
 - ระบบสุ่มจับคู่สนทนาแบบ 1 ต่อ 1 ที่จับคู่ได้ตามการเลือกประเทศและความสนใจของผู้ใช้
 - สามารถส่งข้อความ รูปภาพ วิดีโอ สติกเกอร์
3. ห้องแชทแบบกลุ่ม (Group Chat Room)
 - สร้างห้องแชทหรือเข้าร่วมห้องแชท (สาธารณะหรือส่วนตัว)
 - การตั้งรหัสหรือใส่รหัส 4 หลัก เพื่อสร้างหรือเข้าร่วม  (เฉพาะห้องส่วนตัว)
 - เชิญเพื่อนเข้าร่วมห้อง
 - ห้องแบบกลุ่มรองรับไม่เกิน 10 คน
 - ส่งข้อความ รูปภาพ วิดีโอ สติกเกอร์
4. ระบบเพื่อน (Friend System)
 - เพิ่มเพื่อน ลบเพื่อน ปักดาวเพื่อน และบล็อกเพื่อน
 - ดูสถานะออนไลน์ ออฟไลน์
 - ส่งข้อความส่วนตัวกับเพื่อน
 - ดูข้อมูลเพื่อน หรือผู้ใช้ที่ไม่ใช่เพื่อน
# ฟังก์ชันรองของระบบ
1. ระบบความปลอดภัยและการกลั่นกรองเนื้อหา (Safety & Moderation)
 -ตรวจจับคำหยาบก่อนส่งทุกครั้ง
 - แสดงรายการห้องแชทเป็นที่นิยมในหน้าหลัก
 - ลบห้องแชทที่ไม่มีผู้ใช้อยู่ (Auto Cleanup)
 - ข้อมูลแชทส่วนตัวไม่หายเมื่อปิดห้อง
2. การแจ้งเตือน (Notification System)
 - แจ้งการชวนเข้าร่วมห้องแชท
 - แจ้งเตือนจากเพื่อน เช่น ข้อความใหม่ที่เพื่อนส่งมา
3. ฟังก์ชันรองของผู้ใช้
 - ดูจำนวนผู้ใช้ในห้องแชท
 - ระบบแสดงอวตารและไอเท็มตกแต่ง
 - การจัดการ UI ของโปรไฟล์ (เช่น แสดง theme, ไอคอนตกแต่ง)

## เทคโนโลยีที่ใช้
1. Visual Studio Code (VS Code) ใช้เป็นโปรแกรมหลักสำหรับเขียนและแก้ไขโค้ด
2. React.js และ  Vite สำหรับพัฒนา Frontend ให้โหลดเร็วและมีประสิทธิภาพสูง
3. Tailwind CSS สำหรับจัดการสไตล์และการออกแบบหน้าเว็บ
4. Figma สำหรับออกแบบ UI/UX และสร้างต้นแบบ (Prototype)
5. Node.js สำหรับพัฒนา Backend และ API
6. PostgreSQL เป็นฐานข้อมูลแบบเชิงสัมพันธ์อ็อบเจ็กต์ สำหรับจัดเก็บข้อมูลผู้ใช้และข้อมูลกิจกรรม
7. Ibis Paint สำหรับออกแบบตัวละคร (Avatar) และของตกแต่ง
8. Google Chrome ใช้สำหรับทดสอบเว็บแอปพลิเคชันและตรวจสอบการทำงาน
9. GitHub สำหรับเก็บรักษาโค้ดและควบคุมเวอร์ชันของโครงการ
